# JavaScript - React - Rollup

This example demonstrates how to use the Document Markdown Reader library with React and Rollup bundler.

## Try It Online

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/Interview-Challenge-Archive/document-markdown-reader/tree/main/examples/javascript/react/rollup)

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

This will start Rollup in watch mode. Open index.html in your browser.

## Build

```bash
npm run build
```

## Usage

The application provides a file upload interface that supports:
- PDF (.pdf)
- Word (.docx)
- OpenDocument (.odt)
- Pages (.pages)
- Rich Text (.rtf)
- HTML (.html)
- Markdown (.md)
- Plain text (.txt)

Select a file to see its contents converted to Markdown.
