# TypeScript React Webpack Example

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/Interview-Challenge-Archive/document-markdown-reader/tree/main/examples/typescript/react/webpack)

This example demonstrates how to use the DocumentMarkdownReader library in a TypeScript React application with Webpack as the bundler.

## Installation

```bash
npm install
```

## Development

```bash
npm start
```

## Build

```bash
npm run build
```

## Supported File Formats

- PDF (`.pdf`)
- Word (`.docx`)
- OpenDocument (`.odt`)
- Pages (`.pages`)
- Rich Text Format (`.rtf`)
- HTML (`.html`)
- Markdown (`.md`)
- Plain text (`.txt`)

## Usage

The example provides a file upload interface that allows users to select documents and converts them to markdown format.
