# TypeScript Vanilla Webpack Example

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/Interview-Challenge-Archive/document-markdown-reader/tree/main/examples/typescript/vanilla/webpack)

This example demonstrates how to use the `document-markdown-reader` library in a vanilla TypeScript project with Webpack.

## Installation

```bash
npm install
```

## Development

```bash
npm start
```

## Build

```bash
npm run build
```
