# TypeScript + Vue + Rollup

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/Interview-Challenge-Archive/document-markdown-reader/tree/main/examples/typescript/vue/rollup)

This example demonstrates how to use `document-markdown-reader` with Vue 3 and TypeScript, bundled with Rollup.

## Features

- Vue 3 with Composition API
- TypeScript
- Rollup bundler
- Document import support for PDF, DOCX, ODT, Pages, HTML, Markdown, TXT, RTF

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

This will start the development server.

## Build

```bash
npm run build
```

The built files will be in the `dist` folder.

## Usage

Select a document file using the file input. The document content will be converted to Markdown and displayed below.
